# 天気に振り回される移動、振り回されない移動

## 序論

雨の日、タクシーがなかなか捕まらない。「雨だとみんなタクシーに乗りたがる」とよく言われる。この身近な感覚を出発点に、天気は人の移動をどれだけ動かすのかを調べた。

本研究が立てた問いは、天気に反応する移動と反応しない移動があるとすれば、その二つを分けるものは何か、である。仮説は「予定を変えられる移動かどうか」。通勤は雨でも行かなければならないが、遊びは雨なら取りやめられる。予定を変えられる移動ほど天気に振り回されるはずだ、と考えた。これを裁量性仮説と呼ぶ。

## 本論

### 第一段階：タクシーは天気に反応しなかった

まず、曜日や季節といった「いつものパターン」を差し引き、残ったブレ（残差）が降水量とどう連動するかを調べた。この残差化により、天気そのものによる変動だけを取り出せる。

ところが何年分ものデータを見ても、タクシーの利用回数と降水量のあいだにはっきりした関係は現れなかった。連動係数は −0.024 で、統計的にも誤差の範囲としか言えなかった。雨はタクシーをほとんど動かしていなかった。

### 第二段階：自転車は天気に強く反応した

では天気は移動に関係ないのか。同じ手法をシェアサイクルに適用すると、結果は逆だった。自転車の利用は降水量とはっきり負に連動し、連動係数は −0.515、統計的にも有意だった。タクシー（−0.024）と自転車（−0.515）、同じ街・同じ天気・同じ指標で測って、これだけの差が出た。

### 第三段階：なぜ差が出るのか — 裁量性仮説の検証

同じ条件で反応がこれほど違う理由を、裁量性仮説で説明できるか検証した。自転車データを三つの軸で切り分ければ確かめられるはずだった。往復か片道か、平日朝か休日昼か、会員か非会員か。予定を変えられる側（片道・休日昼・非会員）ほど天気に弱ければ、仮説は支持される。三つすべてが一致すれば証明になるはずだった。

しかし一致したのは一つだけだった。往復／片道の軸は仮説と逆向きに出た。会員／非会員の軸は、そもそも仮説を思いついた元データであり、検証には使えないと判明した。結局、支持したのは「時間帯」の軸のみ。さらに問題として、自転車のデータだけでは、天気に弱い理由が「濡れるから（曝露）」なのか「やめられるから（裁量性）」なのか区別できなかった。

### 第四段階：地下鉄を加えて二つの効果を分離

この区別をつけるため、地下鉄データを追加した。地下鉄は濡れないので、もし反応の差が「濡れるから」だけなら、地下鉄は天気に反応しないはずだ。

自転車（濡れる）と地下鉄（濡れない）を、通勤とレジャーの時間帯で並べて比較した。

|  | 通勤 | レジャー |
|---|---|---|
| 自転車（濡れる） | W −0.451 | W −0.667 |
| 地下鉄（濡れない） | W −0.078 | W −0.465 |

横方向に見ると、自転車も地下鉄も、通勤よりレジャーのほうが天気に弱かった（裁量性の効果）。縦方向に見ると、濡れる自転車のほうが濡れない地下鉄より天気に弱かった（曝露の効果）。統計的に検定すると縦の差も横の差もどちらも有意で、その大きさもかなり近かった。つまり「やめられるか（裁量性）」と「濡れるか（曝露）」の両方が、本物の効果として同時に効いていた。

## 結論

天気予報を見て予定を変えられる移動と、変えられない移動がある。この裁量性が、天気に振り回されるかどうかを分ける一因である。ただしそれだけではなく、「濡れるかどうか」という曝露も同じくらい効いていた。

当初期待した「裁量性ですべて説明できる」というきれいな一発の答えにはならなかった。しかし、仮説が部分的にしか支持されなかったことを隠さず、二つの効果を分離してそれぞれが本物だとそこまで統計的に確かめた点に、この研究の意義がある。

今後の課題として、地下鉄の通勤時間帯はまだデータが薄く、反応の有無を言い切れない。ここを厚くして確証を得たい。
